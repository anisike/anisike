HTML basics

What is HTML?

  1. HTML - no programming language, but a Markup Language defining the structure of the content
  2. HTML - consists of series of Elements

  3. ELEMENT - Can have extra information called attributes. Here class is an attribute.

     <p class = "bigFont">its a paragraph</p>

  4. HTML - Nesting Elements

     <p>My cat is <strong>very</strong> grumpy.</p>
  5. HTML - Block vs Inline Elements
    1. Block level elements form a visible block on a page. Appears in new live from the preceding content. Usually Structural Elements like paragraph, lists, navigation menus, footers etc,. Block level element can’t be in inline element, but another block level element can have a nested block level element.
    2. Inline elements are contained in a Block level elements, and surround only small part of the document’s content. An inline element will not cause a new line to appear in the document. It is typically used with text. Eg.
    <a>, <em>, <strong></strong>

  6. ELEMENT - Empty Element

     <img src="images/firefox-icon.png" alt="My test image">

  7. Boolean Attributes

    <input type="text" disabled>
  8. Omitting Quotes in Attributes
    1. Do not omitt if there are multiple words. It might not give desired result. Best practice, always have it.

  9. HTML - Anatomy

    <!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>My test page</title>
</head>
<body>
    <img src="images/firefox-icon.png" alt="My test image">
</body>
</html>
    • ’<!DOCTYPE html>’ - To tell it’s HTML document
    • ‘<html></html>’ - wraps the whole page, sometimes called ‘root’ element.
    • ‘<head></head>’ - things that go in here are not visible to viewers but to the browser. stores ‘title’ ‘meta’ includes Keywords, CSS reference
    • ‘’ - to tell the browser that the page uses utf-8 charset.
    • ‘My test page’ - Shows in browser tab and saved in this name when bookmarked and favourited.
    • ‘<body></body>’ - Everything viewer sees is from this tag.
