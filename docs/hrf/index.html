HRF Incident FAQ

  1. How to check the HRF version in Hana DB? 
    select * from M_XS_APPLICATIONS where DELIVERY_UNIT = 'HCO_RULE_FW';

    OR

    select * from _sys_repo.delivery_units where delivery_unit = 'HCO_RULE_FW';
  2. How to check whether DU is Active or not? (Result should be empty) 
    select distinct t1.package_id 
from _sys_repo.package_catalog as t1 
inner join _sys_repo.inactive_object as t2 
on t1.package_id = t2.package_id 
where t1.delivery_unit = 'HCO_RULE_FW';
  3. How to check HRF Plugin is enabled or not? (result should be true) 
    select value 
from sys.m_inifile_contents 
where file_name = 'indexserver.ini' 
and section = 'repository'
and key = 'jsplugin_sap/hrf/resources/common/runtimeplugin/src/hrfresourceactivation.xsjs';
