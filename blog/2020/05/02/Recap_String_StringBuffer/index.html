Creating of Strings

String s = "It is a String";
String another = new String("Another String");

There is a subtle difference. The first one uses String Pool.

Concatenation of String

Rules

  1. If both operands are numeric, + means addition.
  2. If either operant is String, + means Concatenation.
  3. The expression is evaluated left to right.

Examples

System.out.println(1+2); //3
System.out.println(1+"2"); //12
System.out.println("a"+"b"+3); //ab3
System.out.println(1+2+3+"4"); //64

int three = 3;
String four = "4";
System.out.println(1+2+three+four); //64

String s = "1";
s += 2; //12
s += 3; //123

You know String is Immutable. It is final class and all it’s methods return a new String.

String s = "1";
String s1 = s.concat("2");
s1.concat("3");
System.out.println(s1); //What does it print??

String Pool

For performance improvement java creates a pool of String literals that are used in the Application. A String literal String s = “A Literal”. If already this literal is there in the pool, it is allocated to variable s. But in case of String s = new String(“A new Object”) an actual new String object is created solely for this variable.

Important String Methods

Just remember

  • String is a sequence of characters.
  • Java counts from Index 0.
a n i m a l s
0 1 2 3 4 5 6

length()

String s = "animals";
System.out.println(s.length()); //7

charAt()

method signature is

char charAt(int index)

So

String s = "animals";
System.out.println(s.charAt(0)); // a
System.out.println(s.charAt(5)); // l
System.out.println(s.charAt(7)); // StringArrayIndexOutOfBounds Exception. Trys to access 8th char but string has only 7 characters

indexOf()

The method indexOf() looks at the characters in the string and finds the first index that matches the desired value. indexOf can work with an individual character or a whole String as input. It can also start from a requested position. The method signatures are as follows:

int indexOf(char ch)
int indexOf(char ch, index fromIndex)
int indexOf(String str)
int indexOf(String str, index fromIndex)

Let’s see the code example..

String string = "animals";
System.out.println(string.indexOf('a')); // 0
System.out.println(string.indexOf("al")); // 4
System.out.println(string.indexOf('a', 4)); // 4
System.out.println(string.indexOf("al", 5)); // -1

Method starts from the fromIndex (if not given assumes as 0) and tries to find the whole String given. Returns the position of that if found else -1. No Exception is thrown.

subString()

This returns a subset of the String by slicing from ‘beginIndex’ (inclusive) till ‘endIndex’ (exclusive). Throws exception if the slicing is not possible eighther the given index is wrong or crosses the limit or length of the string.

method signature is as below

String substring(int beginIndex)
String substring(int beginIndex, int endIndex)

String string = "animals";
System.out.println(string.substring(3)); // mals
System.out.println(string.substring(string.indexOf('m'))); // mals
System.out.println(string.substring(3, 4)); // m
System.out.println(string.substring(3, 7)); // mals

//Some more
System.out.println(string.substring(3, 3)); // empty string
System.out.println(string.substring(3, 2)); // throws exception
System.out.println(string.substring(3, 8)); // throws exception

toLowerCase() and toUpperCase()

Singnature

String toLowerCase(String str)
String toUpperCase(String str)

String string = "animals";
System.out.println(string.toUpperCase()); // ANIMALS
System.out.println("Abc123".toLowerCase()); // abc123

equals() and equalsIgnoreCase()

String equals(String str)
String equalsIgnoreCase(String str)

System.out.println("abc".equals("ABC")); // false
System.out.println("ABC".equals("ABC")); // true
System.out.println("abc".equalsIgnoreCase("ABC")); // true

startsWith() and endsWith()

The startsWith() and endsWith() methods look at whether the provided value matches part of the String. The method signatures are as follows:

boolean startsWith(String prefix)
boolean endsWith(String suffix)

System.out.println("abc".startsWith("a")); // true
System.out.println("abc".startsWith("A")); // false
System.out.println("abc".endsWith("c")); // true
System.out.println("abc".endsWith("a")); // false

contains()

similar to startsWith() and endsWith() but just does’nt care start or end. It should match anywhere.

boolean contains(String str)

System.out.println("abc".contains("b")); // true
System.out.println("abc".contains("B")); // false

replace()

searches the string/char and replaces everywhere.

String replace(char oldChar, char newChar)
String replace(CharSequence oldChar, CharSequence newChar)

System.out.println("abcabc".replace('a', 'A')); // AbcAbc
System.out.println("abcabc".replace("a", "A")); // AbcAbc

trim()

The trim() method removes whitespace from the beginning and end of a String.

public String trim()

System.out.println("abc".trim()); // abc
System.out.println("\t a b c\n".trim()); // a b c

Method chaining

String result = "AniMaL ".trim().toLowerCase().replace('a', 'A').toUpperCase();
System.out.println(result); //ANIMAL

This is just a series of String methods. Each time one is called, the returned value is put in a new variable. There are five String values along the way, and ANIMAL is output.
